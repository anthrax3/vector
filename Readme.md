
# point

  `Point` object for canvas etc.

## Installation

```
$ component install component/point
```

## Example

```js
var Point = require('point');
var a = new Point(5, 5);
var b = new Point(10, 10);
a.distance(b);
```

  or without `new`:

```js
var p = require('point');
var a = p(5, 5);
var b = p(10, 10);
a.distance(b);
```
